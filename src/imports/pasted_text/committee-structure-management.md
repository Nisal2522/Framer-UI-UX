8.	Asset 
This module provides a complete digital lifecycle management system for all assets owned by, donated to, or used by an AC. It ensures transparency, accountability, and proper stewardship of cooperative resources.
Allow a button on top of the page to register a new asset. Add New window should consist of the following: 
a)	Asset type (Please select from the dropdown list: Equipment, Vehicle, Building, Infrastructure, or Other.)
b)	Asset name
c)	Description
d)	Serial/identification number
e)	Acquisition date
f)	Acquisition method (Please select from the dropdown list: Purchased/Donated by PEARL/Donated by Other Program/Government Grant/Own Funds)
g)	Acquisition cost or estimated value
h)	Current location
i)	Responsible custodian name
j)	Allow attachment per asset for visual documentation of the asset at registration and after maintenance
Prompt 1:
For the Asset page, the following tabs should be included:
a)	Asset Inventory
b)	Usage Log
c)	Disposal Request
Asset Inventory
Prompt 2:

Additionally, please tag and filter assets specifically funded or donated by the PEARL project for separate reporting to the FAO/GCF.
