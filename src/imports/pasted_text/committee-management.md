Asset  

This module provides a complete digital lifecycle management system for all assets owned by, donated to, or used by an AC. It ensures transparency, accountability, and proper stewardship of cooperative resources. 

Allow a button on top of the page to register a new asset. Add New window should consist of the following:  

Asset type (Please select from the dropdown list: Equipment, Vehicle, Building, Infrastructure, or Other.) 

Asset name 

Description 

Serial/identification number 

Acquisition date 

Acquisition method (Please select from the dropdown list: Purchased/Donated by PEARL/Donated by Other Program/Government Grant/Own Funds) 

Acquisition cost or estimated value 

Current location 

Responsible custodian name 

Allow attachment per asset for visual documentation of the asset at registration and after maintenance 

Prompt 1: 

For the Asset page, the following tabs should be included: 

Asset Inventory 

Usage Log 

Disposal Request 

Asset Inventory 
Prompt 2: 

In the Asset Inventory tab, please display all added assets in a list view, as there will be a substantial number of assets under each AC. The asset inventory should include filters for asset type, condition, acquisition source, and date range, and it should be exportable to both PDF and Excel formats.  

Additionally, please tag and filter assets specifically funded or donated by the PEARL project for separate reporting to the FAO/GCF. Newly registered assets must go through a verification step by the Commune Officer before being marked as "Verified" in the system. Inform the user that once the commune officers approve it, it will be displayed in the asset inventory.
